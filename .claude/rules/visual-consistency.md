# Visual Consistency — Designkontrakt

Alle visuelle komponenter i en visualisering skal dele:
- Samme spacing-skala (multipla af base-unit, fx 8px)
- Samme farvekontrakt (definér roller FØR implementering)
- Samme typografi-skala (definér h1-h4, body FØR implementering)
- Samme border/shadow-sprog
- Samme interaktions-feedback (hover, transitions, fokus)

Før multi-komponent visualisering — definér:
1. Spacing: base unit og skala
2. Farver: palette med roller
3. Typografi: font-stack og størrelses-skala
4. Afrunding: border-radius værdier
5. Skygger: elevation-niveauer (eller ingen)
