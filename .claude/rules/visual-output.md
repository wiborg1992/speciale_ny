# Visual Output-kvalitet

Alle HTML-visualiseringer skal:
- Være komplet renderbare uden eksterne afhængigheder
- Inkludere inline CSS (ingen eksterne stylesheets)
- Have fallback-indhold hvis data mangler
- Bruge semantisk HTML5 (section, article, header, main)
- Være responsive (viewport meta + media queries)
- Aldrig bruge noter eller pseudokode som output

Verificér at HTML er valid og renderbar før du præsenterer den.
